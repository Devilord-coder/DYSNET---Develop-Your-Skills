# Сайт DYSNET - развивай свои навыки
## Идея проекта
Разработка сайта для развития современных навыков.
## Задачи проекта
Задачи, которые требуются для реализации проекта:
* Изучить библиотеку Flask;
* Спроектировать архитектуру взаимодействия;
* Разработать систему регистрации и авторизации;
* Спроектировать несколько модулей навыков.
## Описание работы проекта
В начале пользования сайтом пользователю необходимо
зарегистрироваться, далее он будет входить в свой аккаунт. После
регистрации у пользователя будет возможность выбора навыка или перехода в
свой профиль, где будет хранится статистика всех навыков.
## Актуальность
Каждый человек постоянно чему-то обучается. Для этого ему
необходимо искать разные источники информации и приложения/сайты для
их тренировки. Наш сайт поможет удобнее развивать большое количество
навыков за счёт того, что они собраны на одном сайте.
